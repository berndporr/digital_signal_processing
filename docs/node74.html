Fixed point IIR filters

Fig. 21 shows the implementation of a fixed point IIR filter. It's a Direct Form I filter with one accumulator so that temporary overflows can be compensated. The original floating point IIR coefficients are scaled up by factor so that they max out the range of the integer coefficients.
